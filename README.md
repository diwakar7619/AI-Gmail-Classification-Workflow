# AI Gmail Classification Workflow

An AI-powered Gmail automation workflow built using n8n, OpenAI, and Gmail API that automatically classifies incoming emails into trusted and review-required categories.

---

## Features

- Real-time Gmail monitoring
- AI-based email classification
- Automatic Gmail label assignment
- Trusted sender detection
- Review-required filtering
- Structured AI response parsing
- Clean automated workflow

---

## Workflow Overview

1. Gmail Trigger monitors incoming emails
2. OpenAI analyzes email content
3. Structured Output Parser formats AI response
4. Workflow checks trust conditions
5. Gmail labels are automatically applied:
   - Trusted
   - Needs Review

---

## Tech Stack

- n8n
